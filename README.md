# Fitness retreat — одностраничный сайт

Лёгкий проект: статическая одностраничная витрина с простым Node.js бэкендом для сохранения заявок.

Запуск локально:

```bash
npm install
npm start
# или для разработки с авто-перезапуском
npm run dev
```

Сайт будет доступен по адресу `http://localhost:3000`.

Файлы:
- `public/` — фронтенд (index.html, styles.css, script.js)
- `server.js` — простой Express-сервер
- `data/contacts.json` — файл с сохранёнными заявками

Дальше можно: подключить SMTP/Telegram для уведомлений, стилизовать дизайн или задеплоить на хостинг.

## Как получить публичную ссылку

### Рекомендуемый вариант — Render.com
1. Создай репозиторий на GitHub и закоммить туда все файлы проекта.
2. Перейди на https://render.com и зарегистрируйся.
3. Создай новый Web Service, привязав его к репозиторию.
4. В поле Build Command установи `npm install`.
5. В поле Start Command установи `npm start`.
6. Добавь переменные окружения, если хочешь использовать SMTP для отправки почты:
   - `SMTP_HOST`
   - `SMTP_USER`
   - `SMTP_PASS`
   - `SMTP_PORT`
   - `SMTP_SECURE`
   - `CONTACT_EMAIL`
7. Нажми Deploy.

После успешного деплоя сайт будет доступен по публичному URL вида `https://<имя-проекта>.onrender.com`.

### Быстрый вариант — Netlify / GitHub Pages (frontend только)
Если нужны только страница и визуальный лендинг без отправки почты на твой сервер, можно разместить `public/` через Netlify или GitHub Pages.

### Локальный предварительный просмотр
```bash
cd public
python3 -m http.server 8000
```
Открой `http://localhost:8000`.
